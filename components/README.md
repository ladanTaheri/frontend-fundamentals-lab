# Components

A collection of reusable UI components built with HTML and CSS for frontend development practice and reference.

## Included Examples

- Authentication forms
- Custom buttons
- Navigation menus
- Styled tables
- Responsive footer

## Technologies

- HTML5
- CSS3
- Flexbox

## Purpose

This section contains reusable frontend components commonly used in web interfaces. The examples focus on structure, styling, responsiveness, and modern CSS techniques while reinforcing component-based UI development.
