
# Minimal Login UI

A clean and minimal login form interface with gradient background styling and centered layout design.

## Features

- Gradient page background
- Minimal card-style login form
- Styled form inputs
- Interactive button hover effect
- Simple and modern UI design
- Responsive centered layout using Flexbox

## Technologies

- HTML5
- CSS3
- Flexbox

## Learning Topics

- Form structure
- Input styling
- Flexbox alignment
- Linear gradients
- Hover effects
- UI spacing and layout techniques

## Goal

Practicing modern login form design and building reusable authentication UI components with pure HTML and CSS.
