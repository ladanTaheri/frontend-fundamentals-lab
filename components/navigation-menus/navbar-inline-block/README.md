

# Simple Navbar (Inline-Block)

A simple horizontal navigation bar implemented using HTML and CSS with the inline-block layout technique.

## Features

- Horizontal navigation structure
- Active menu item styling
- Hover effects
- Icon integration
- Dark-themed UI

## Technologies

- HTML5
- CSS3
- Font Awesome

## Overview

This example demonstrates a basic navigation menu layout using inline-block elements and classic CSS styling techniques.
