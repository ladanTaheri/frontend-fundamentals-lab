# Responsive Footer

A responsive footer component built with Flexbox.

## Features

- Multi-column footer layout
- Contact information section
- Quick access links
- Logo area
- Responsive Flexbox structure
- Hover effects for links

## Technologies

- HTML5
- CSS3
- Flexbox

## Purpose

This project demonstrates how to build a responsive footer component using modern Flexbox techniques. The layout automatically adapts to different screen sizes while maintaining a clean and flexible structure.
