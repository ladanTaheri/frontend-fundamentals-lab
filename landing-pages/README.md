# Landing Pages

A collection of landing page projects built as part of frontend development learning and practice.

## Included Examples

- Transit Landing Page
- Newsbit Landing Page
- Travelo Landing Page


## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Flexbox
- CSS Grid
- CSS Variables
- Font Awesome
- Google Fonts
- Responsive Design

## Purpose

This section focuses on learning and practicing modern frontend development through real-world landing page projects. The examples reinforce layout design, responsive implementation, semantic HTML, and reusable CSS techniques.
