# NewsBit Landing Page - Grid Version

A **CSS Grid-based implementation** of the NewsBit landing page, focused on modern layout design and responsive UI structuring.

This version demonstrates how complex landing page layouts can be efficiently built using CSS Grid for full-page structure control.

---

## 📌 Overview

This project is a Grid-based version of the NewsBit landing page originally implemented using Flexbox.

It provides a structured comparison between:

- Flexbox layout approach
- CSS Grid layout approach

---

## 🧱 Key Features

- Fully responsive layout using CSS Grid
- Advanced use of `grid-template-areas`
- Multi-section hero layout design
- Clean separation of layout and components
- Mobile-first responsive adjustments

---

## 🆚 Flexbox vs Grid Version

This repository contains both versions for comparison purposes:

- **Flexbox version:** focuses on one-dimensional layout flow
- **Grid version:** focuses on full-page two-dimensional layout structure

Both implementations share the same design but use different layout strategies.

---

## 📁 Structure

This version includes:

- Hero section redesigned with CSS Grid
- Responsive breakpoints for different screen sizes
- Reusable card-based layout structure

---

## 🎯 Purpose

This project is part of a front-end development learning track focused on mastering modern CSS layout systems and building production-like UI structures.
