## Transit Landing Page 

A responsive landing page built with HTML5 and CSS3 as part of the Frontend Fundamentals Lab collection.



## Features

- navigation bar
- Hero section
- Feature cards section
- Testimonials section
- Contact form
- footer
- Mobile-first responsive design
- Flexbox layout



## Technologies

- HTML5
- CSS3
- Flexbox
- Font Awesome



## Learning Objectives

- Practice semantic HTML structure
- Build responsive layouts with Flexbox
- Organize reusable CSS classes
- Improve component-based page structure
- Convert a non-responsive design into a responsive implementation
