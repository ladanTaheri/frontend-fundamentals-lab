## Travelo Landing Page
A responsive travel landing page built as part of the Frontend Fundamentals repository.

This project recreates the layout of a modern travel website to strengthen practical front-end development skills through building a real-world interface from scratch.

## This project focuses on
- Semantic HTML5
 - Responsive Design
- Mobile-First Development
- Flexbox Layouts
- CSS Grid Layouts
- Reusable Components
- CSS Variables
- Vanilla JavaScript
- DOM Manipulation
- Responsive Navigation
- Features
- Responsive navigation menu
- Mobile navigation toggle
- Hero banner section
- Travel search section
- Popular destinations section
- Newsletter subscription section
- Popular places cards
- Video banner section
- Responsive footer
- Interactive hover effects
- Responsive layouts for mobile, tablet, and desktop

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- Flexbox
- CSS Grid
- CSS Variables
- Font Awesome
- Google Fonts

## Learning Goals
This project was created to improve practical front-end development skills by recreating a real-world landing page using semantic HTML, modern CSS, responsive design techniques, and vanilla JavaScript without relying on CSS frameworks.
