# Responsive Flex Layout

A responsive page layout built using Flexbox and modern CSS techniques.

## Features

- Responsive header section
- Flexbox-based banner layout
- Responsive image gallery
- Footer navigation
- Social media section
- Mobile-first design approach

## Technologies

- HTML5
- CSS3
- Flexbox
- Media Queries

## Purpose

This project demonstrates how responsive page layouts can be built using Flexbox. It focuses on content alignment, flexible layout structures, and adapting page sections across different screen sizes.
