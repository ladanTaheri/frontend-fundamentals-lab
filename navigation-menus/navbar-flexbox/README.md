# Flexbox Navbar

A navigation menu implementation built with HTML and CSS using Flexbox for layout and alignment.

## Features

- Horizontal navigation layout
- Flexbox-based alignment
- Icon and text navigation items
- Active menu state styling
- Hover interactions
- Font Awesome integration

## Technologies

- HTML5
- CSS3
- Flexbox
- Font Awesome

## Purpose

This example demonstrates how Flexbox can be used to create responsive-friendly navigation menus with aligned icons, interactive states, and clean horizontal layouts.
